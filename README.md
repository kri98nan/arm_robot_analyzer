# arm_robot_analyzer
Forward kinemaics analysis and demonstration of DH-Parameters upto 7 joint manipulator

Libraries:
```
pip install Pillow
pip install PyOpenGL
pip install pygame
```

Command:
```
python Manipulator_Analyzer.py
```
